# 🍕 Pizza House Dashboard

A modern, full-featured Admin Dashboard for managing a Pizza House website.  
Built with PHP (OOP), MySQL, HTML, CSS, and JavaScript.

## 📌 Features

- 🔐 Admin Authentication (Login / Logout)
- 📊 Dashboard Overview (Stats & Analytics)
- 🍕 Manage Pizzas (Add / Edit / Delete)
- 🛒 Manage Orders
- 👥 Manage Users
- ❤️ Wishlist Management
- 🗂 Category Management
- 🧾 Order Status Control
- 💾 Secure PDO Database Connection
- ⚠️ Custom Database Error Page
- 🎨 Modern UI Design
- 📱 Responsive Layout
